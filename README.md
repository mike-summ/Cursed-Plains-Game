# Cursed Plains
A Unity 3D game by Michael Summers

Defeat the two bosses to win the game. Walking through the blue flowers will set the respawn location at that point.

## Play
https://mike-summ.github.io/Cursed-Plains-Game/
## Controls

- WASD: Move
- Space: Dash
- LMB: Fireball
- RMB: Primal Fireball
